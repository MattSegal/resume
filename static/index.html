Online Poker Image Recognition Bot (Github)

I developed a Python Script that reads data from an online poker table, as a proof-of-concept for an automated online poker bot. I also made a small GUI that allows users to interact with the script. The script works as follows, it:

  • Locates and isolates areas of the screen containing desirable data
  • Processes these smaller images, removing noise
  • Breaks images of numbers up into single digits
  • Classifies each digit using logistic regression
  • Returns a numerical value for each image
